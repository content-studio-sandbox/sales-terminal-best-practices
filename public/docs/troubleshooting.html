Troubleshooting Guide

Common Issues and Solutions

This guide helps you resolve common issues when using Your Projects at IBM.

Authentication Issues

Cannot Log In

  • Symptoms
  • Solutions
  • Redirected to login page repeatedly
  • “Authentication failed” error
  • Stuck on loading screen

Clear Browser Cache 1. Open browser settings 2. Clear browsing data 3. Select “Cookies and site data” 4. Clear and try again

Try Incognito/Private Mode 1. Open incognito/private window 2. Navigate to platform 3. Attempt login 4. If successful, clear cache in normal mode

Verify IBM Credentials 1. Test credentials at ibm.com 2. Reset password if needed 3. Ensure account is active 4. Contact IT if account locked

Check Browser Compatibility - Use Chrome 90+, Firefox 88+, Safari 14+, or Edge 90+ - Update browser to latest version - Disable browser extensions temporarily

Session Expires Frequently

Cause: Short session timeout or network issues

Solutions: - Stay active on the platform - Refresh page periodically - Check network stability - Contact admin to adjust timeout

BYOK Configuration Issues

Test Connection Fails

  • WatsonX.ai Connection
  • Watson Orchestrate Connection

Error: “Invalid API key”

Check: - API key copied correctly (no spaces) - Key hasn’t expired - Key has required permissions - Using correct IBM Cloud region

Fix: 1. Generate new API key in IBM Cloud 2. Copy entire key carefully 3. Paste in settings 4. Test again

Error: “Project not found”

Check: - Project ID is correct - Project exists in watsonx.ai - You have access to project - Project is in same region as API key

Fix: 1. Open watsonx.ai 2. Navigate to project 3. Copy Project ID from settings 4. Paste in platform 5. Test again

Error: “Invalid credentials”

Check: - API key is correct - Assistant ID is correct - Assistant is active - You have access to assistant

Fix: 1. Verify credentials in Orchestrate console 2. Generate new API key if needed 3. Copy Assistant ID 4. Update in platform 5. Test again

Error: “Service unavailable”

Check: - Watson Orchestrate service is running - No scheduled maintenance - Network connectivity - Firewall not blocking requests

Fix: 1. Check IBM Cloud status page 2. Wait if maintenance ongoing 3. Try from different network 4. Contact IBM support

Credentials Not Saving

Symptoms: - Settings revert after save - “Save failed” error - Credentials disappear

Solutions:

Check Browser Settings - Allow cookies for the site - Disable “Block third-party cookies” - Allow local storage

Try Different Browser - Test in another browser - If works, clear cache in original browser

Check Network - Ensure stable connection - Try from different network - Check firewall settings

Project Management Issues

Cannot Create Project

  • Missing Required Fields
  • Permission Denied

Error: “Please fill all required fields”

Required Fields: - Project name - Description (minimum 50 characters) - Required skills - Team size - Duration

Fix: 1. Review form carefully 2. Fill all required fields 3. Ensure description is detailed 4. Add at least one required skill 5. Submit again

Error: “You don’t have permission to create projects”

Cause: Your role doesn’t allow project creation

Fix: 1. Verify your role (check header badge) 2. Contact admin if role is incorrect 3. Request permission upgrade if needed

Project Not Appearing

Symptoms: - Created project doesn’t show in list - Project missing from dashboard

Solutions:

Refresh the Page - Hard refresh (Ctrl+Shift+R or Cmd+Shift+R) - Clear cache and reload

Check Filters - Remove any active filters - Check status filter (open/closed) - Clear search terms

Verify Creation - Check for confirmation message - Look in “All Projects” tab - Search by project name

Application Issues

Cannot Submit Application

  • Profile Incomplete
  • Project Closed
  • Already Applied

Error: “Please complete your profile”

Required: - Personal information - Skills - Resume uploaded

Fix: 1. Go to profile settings 2. Fill all required fields 3. Upload resume (PDF or DOCX) 4. Save profile 5. Try applying again

Error: “This project is no longer accepting applications”

Cause: Application deadline passed or project filled

Fix: - Browse other open projects - Set up alerts for new projects - Contact manager if urgent

Error: “You’ve already applied to this project”

Cause: Duplicate application attempt

Fix: - Check “Your Applications” tab - Track existing application status - Wait for response

Application Status Not Updating

Symptoms: - Status stuck on “Pending” - No notifications received

Solutions:

Check Notification Settings - Verify email notifications enabled - Check spam folder - Update email address if needed

Contact Manager - Use in-app messaging - Send polite follow-up - Ask for timeline

Be Patient - Managers review within 3-5 days - High application volume may delay - Check back regularly

Resume Matching Issues

AI Matching Fails

  • Upload Errors
  • No Results Returned
  • Low Match Scores

Error: “Failed to upload resume”

Check: - File format (PDF or DOCX only) - File size (under 10MB) - File not corrupted - Stable internet connection

Fix: 1. Convert to PDF if needed 2. Compress large files 3. Try different file 4. Upload one at a time

Error: “Failed to match resumes”

Check: - BYOK credentials configured - Test connection successful - Job description provided - At least one resume uploaded

Fix: 1. Verify BYOK settings 2. Test connection 3. Ensure job description is detailed 4. Try with fewer resumes 5. Check error logs

Not an Error: AI provides honest assessment

Improve Results: - Write detailed job descriptions - Include specific skills - Specify experience level - Upload relevant resumes

Dashboard and Analytics Issues

Dashboard Not Loading

  • Blank Dashboard
  • Slow Loading

Symptoms: - White screen - Loading spinner forever - No data displayed

Solutions:

Refresh Page - Hard refresh (Ctrl+Shift+R) - Clear cache - Try incognito mode

Check Browser Console 1. Open developer tools (F12) 2. Check Console tab for errors 3. Screenshot errors 4. Report to support

Try Different Browser - Test in Chrome, Firefox, or Edge - Update browser to latest version

Symptoms: - Dashboard takes long to load - Charts render slowly

Solutions:

Check Network - Test internet speed - Close bandwidth-heavy apps - Try wired connection

Reduce Date Range - Select shorter time period - Load less data at once - Export for offline analysis

Clear Cache - Clear browser cache - Restart browser - Try again

Incorrect Metrics

Symptoms: - Numbers don’t match expectations - Charts show wrong data

Solutions:

Verify Date Range - Check selected date range - Ensure correct time zone - Adjust filters

Refresh Data - Click refresh button - Wait for data sync - Check again in few minutes

Report Issue - Screenshot the issue - Note expected vs actual values - Contact support with details

Watson Orchestrate Chatbot Issues

Chatbot Not Responding

  • No Response
  • Incorrect Responses

Symptoms: - Type message, no reply - Chatbot shows “thinking” forever - Error message displayed

Solutions:

Check BYOK Configuration 1. Open Settings 2. Go to Watson Orchestrate tab 3. Test connection 4. Reconfigure if needed

Restart Chatbot 1. Close chatbot panel 2. Refresh page 3. Open chatbot again 4. Try query again

Simplify Query - Use simpler language - Break into smaller questions - Be more specific - Try example queries

Symptoms: - Chatbot gives wrong information - Doesn’t understand question - Provides irrelevant answers

Solutions:

Rephrase Question - Use different words - Be more specific - Provide context - Try natural language

Use Suggested Queries - Click suggested questions - Follow examples in docs - Learn from successful queries

Report Issue - Note the query - Screenshot response - Submit feedback - Help improve AI

Performance Issues

Slow Page Load

Causes: - Slow internet connection - Large data sets - Browser issues - Server load

Solutions:

Optimize Browser - Close unused tabs - Disable extensions - Clear cache - Update browser

Check Network - Test internet speed - Use wired connection - Avoid peak hours - Try different network

Reduce Data Load - Use filters - Limit results - Paginate large lists - Export for offline work

Timeout Errors

Error: “Request timeout”

Solutions: - Retry the operation - Break into smaller requests - Check network stability - Contact support if persists

Mobile Issues

Mobile Display Problems

Symptoms: - Layout broken on mobile - Buttons not clickable - Text too small

Solutions:

Use Desktop Version - Platform optimized for desktop - Mobile support limited - Use laptop/desktop when possible

Try Landscape Mode - Rotate device - More screen space - Better layout

Update Browser - Use latest mobile browser - Try Chrome or Safari - Clear mobile cache

Getting Additional Help

Before Contacting Support

Gather Information: - What were you trying to do? - What happened instead? - Error messages (screenshot) - Browser and version - Steps to reproduce

Try Basic Troubleshooting: - Refresh page - Clear cache - Try different browser - Check internet connection

Contact Support

Email Support - Email: support@ibm.com - Include screenshots - Describe issue clearly - Provide account details

Slack Channel - Join: #your-projects-support - Ask questions - Get quick help - Share solutions

Submit Ticket - Use in-app feedback - Provide detailed description - Attach screenshots - Track ticket status

Community Resources

Documentation - Browse all guides - Search for keywords - Check FAQ section

Video Tutorials - Watch how-to videos - Follow step-by-step - Learn best practices

User Forums - Ask community - Share experiences - Learn from others

Frequently Asked Questions

Q: Why can’t I see the Executive Dashboard?
A: Only users with Executive role can access it. Check your role badge in the header.

Q: How long does application review take?
A: Typically 3-5 business days. Managers are notified of new applications.

Q: Can I edit my application after submitting?
A: No, but you can contact the manager directly to provide additional information.

Q: Why was my application declined?
A: Check for feedback from the manager. Common reasons: skills mismatch, project filled, timing issues.

Q: How do I delete my account?
A: Contact support@ibm.com to request account deletion.

Q: Can I apply to multiple projects?
A: Yes! Apply to 3-5 projects that genuinely interest you.

Q: How do I report a bug?
A: Use the in-app feedback form or email support@ibm.com with details.
