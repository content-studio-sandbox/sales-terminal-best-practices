KAN-48: Implement Intern Project Tracking System

Overview

Implemented a comprehensive Monday.com-style project tracking system for interns with task management, status tracking, file attachments, and team collaboration features.

  • Status Tracking: Support for Todo, In Progress, On Hold, and Done statuses
  • Due Date Tracking: Visual due date indicators with date picker
  • File Attachments: Upload and download files for each task
