Testing Guide for All Feature Branches

This guide will help you test all the feature branches created today before deploying to OpenShift.

Overview of Branches

Today we created the following feature branches: 1. TASK-10-Task-15: Skills management & matching system 2. TASK-16-Task-20: Learning catalog & verification 3. TASK-21-Task-25: Dashboard analytics & filtering 4. TASK-26-Task-27: Copy updates & accessibility

Testing Strategy

Option 1: Test Each Branch Individually (Recommended)

Test each feature branch separately to isolate issues:

cd your-projects

# Test TASK-10-Task-15 (Skills & Matching)
git checkout TASK-10-Task-15
npm install
npm run dev
# Test features, then stop server (Ctrl+C)

# Test TASK-16-Task-20 (Learning Catalog)
git checkout TASK-16-Task-20
npm install
npm run dev
# Test features, then stop server (Ctrl+C)

# Test TASK-21-Task-25 (Dashboard Analytics)
git checkout TASK-21-Task-25
npm install
npm run dev
# Test features, then stop server (Ctrl+C)

# Test TASK-26-Task-27 (Copy & Accessibility)
git checkout TASK-26-Task-27
npm install
npm run dev
# Test features, then stop server (Ctrl+C)

Option 2: Merge All to a Test Branch

Create a combined test branch with all features:

cd your-projects

# Create test branch from main
git checkout main
git pull origin main
git checkout -b test-all-features

# Merge all feature branches
git merge TASK-10-Task-15
git merge TASK-16-Task-20
git merge TASK-21-Task-25
git merge TASK-26-Task-27

# Resolve any conflicts if they occur
# Then install and test
npm install
npm run dev

Common Issues & Solutions

Issue: npm install fails

Solution:

rm -rf node_modules package-lock.json
npm install

Issue: TypeScript errors

Solution:

npm run build
# Fix any reported errors

Issue: Database migrations fail

Solution:

cd your-projects/supabase
supabase db reset
supabase db push

Issue: Port already in use

Solution:

# Kill process on port 5173
lsof -ti:5173 | xargs kill -9
# Or use a different port
npm run dev -- --port 5174

Quick Test Script

Save this as test-all.sh for quick testing:

#!/bin/bash

branches=("TASK-10-Task-15" "TASK-16-Task-20" "TASK-21-Task-25" "TASK-26-Task-27")

for branch in "${branches[@]}"; do
    echo "Testing $branch..."
    git checkout "$branch"
    npm install
    npm run build
    if [ $? -eq 0 ]; then
        echo "✅ $branch builds successfully"
    else
        echo "❌ $branch has build errors"
    fi
done

git checkout main

Run with: chmod +x test-all.sh && ./test-all.sh
